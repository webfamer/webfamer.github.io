本地环境的搭建

1、node

2、npm

3、express的安装

  • 全局安装：npm install express -g （-g 代表全局安装）
  • npm install -g express-generato (为什么要有这一步呢)

4.X版本，把generato分离出来了，需要单独安装

4、安装数据库

  • npm install mysql -save

  • 打开数据库新建数据库表（这里取的数据库名是user）

  • 在mysql中配置数据库（dbconfig.js）

5、数据库配置好之后,我就来测试一下,找到路由中的(index.js)

6、热更新nodemon

7、这里我把修改增删改查的数据放在了一个cateController.js中继续封装
