块元素和内嵌元素

块元素的特征

  1. 默认独占一行
  2. 没有宽度时，默认排满一行
  3. 支持所有css命令

内联元素特征

  1. 同排可以继续跟同类标签
  2. 内容撑开高度
  3. 不支持宽高
  4. 不支持上下margin
  5. 代码换行被解析
