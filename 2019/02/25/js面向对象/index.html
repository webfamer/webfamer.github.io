js面向对象知识点总结

我们把系统自带的对象叫做系统对象
面向对象编程(OOP)的特点

  1. 抽象：抓住核心问题
  2. 封装：只能通过对象来访问方法
  3. 继承：从已有的对象上继承出新的对象(其实就是去复用代码)
  4. 多态：多对象的不同形态 (其实也是复用代码) 类比一台电脑，有个USB接口，可以连接各种设备

对象的组成

方法(行为，操作) —函数：过程，动态的
属性—-变量：状态，静态的

工厂方式与构造函数

this(隐式返回)
new后面调用的函数：叫做构造函数

原型

原型：去改写对象下面公用的方法或属性，让公用的方法或者属性在内存中存在一份(提高性能)
类比学习：
原型：css中的class
普通方法：css中的style
style的优先级比class高，所以普通方法优先级也比原型高
原型：prototype 写在构造函数下面(即new后面的那个函数) Array.prototype

总的写法：

使用：
var 对象1 = new 构造函数();
对象1.方法();
在js源码中：系统对象也是基于原型的程序
尽量不要去修改或者添加系统对象下面的方法和属性

包装对象

基本类型都有自己的包装对象 ：String Number Boolean

原型链

实例对象与原型对象之间的连接,叫做原型链
原型链的最外层：Object.prototype
