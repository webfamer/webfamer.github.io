正则表达式

发表于 2019-02-11

正则表达式知识点

正则：也叫规则，让计算机能够读懂人类的规则
正则都是操作字符串的
写法：
var re = /a/ 简写
var re = new RegExp(‘a’); 全称写法

##正则表达式的常用方法

###1. test()
test: 正则去匹配字符串，如果匹配成功则返回真，如果匹配失败则返回假
test的写法： 正则.test(字符串)
转义字符：

###2. search()
search: 正则去匹配字符串，如果匹配成功则返回匹配成功的位置，如果匹配失败返回-1
search的写法： 字符串.search(正则)

正则中的默认是区分大小写的,如果不区分大小写，在正则最后加上标识符 i

var re = /a/i 简写
var re = new RegExp(‘a’,’i’); 全称写法

###3. match()
match():正则去匹配字符串，如果匹配成功则返回匹配成功的数组，如果匹配不成功，就返回null
match的写法：字符串.match(正则)

正则默认：正则匹配成功就会结束，不会继续匹配,如果想全部查找，就要加标识 g(全局匹配)

###4.replace()
replace()：正则去匹配字符串，匹配成功的字符去替换掉新的字符串
replace的写法：字符串.replace(正则，新的字符串)

replace()的第二个参数可以是一个字符串，也可以是一个回调函数
replace(正则，function(str){
//回调函数的第一个参数(str)，就是匹配成功的字符
})

案例-点击过滤关键词

效果图

var str = ‘2013-6-7’ ;
var re = /(\d+)(-)/g ;
str.replace(re,function($0){
第一个参数:$0(母亲),
第二个参数：$1(第一个孩子),
第三个参数：$2(第三个孩子)
})
匹配子项： 小括号（ ） （还有另一个意思，分组操作）
把正则的整体叫做(母亲)
然后把左边第一个小括号里面的正则，叫做这个的第一个子项（母亲的第一个孩子）
第二个括号就是第二个孩子

match也有匹配子项：
var str = ‘abc’,
var re = /(a)(b)(c)/;
alert(str.match(re));//abc,a,b,c

//字符类： 一组相似的元素 [ ]中括号的整体代表一个字符
var str = ‘abcd’
var re = /a[bc]d/;
alert(re.test(str));

排除：^ 如果^写在[ ]里面的话，就代表排除的意思

/[a-z1-9A-Z]/ 整体只代表一位 代表多位的话后面更上+

###量词：{ }
{4,7} 字符最少出现4次最多出现7次

{4} 正好出现4次

+：{1，}     \d{1,} 至少出现一次或多次

？：{0,1}    至少出现0次或多次

*：{0，}    至少出现0次

^ :放到正则的最开始位置，就代表起始的意思

$:放到正则的最后位置，就代表结束的意思
